Assignment Management System

A comprehensive assignment management platform integrating multiple technologies across a full-stack application for enhanced functionality.

Project Overview

This project showcases a full-stack approach to building and deploying a scalable web application for managing assignments, integrating Node.js, AWS, and serverless architectures. The system is designed to automate and streamline the handling of educational assignments through sophisticated technical solutions.

Cloud App on GitHub

Node.js Web Application

  • User authentication with secure login and session management.
  • CRUD operations for assignment management.
  • Automated health checks to ensure continuous application performance.
  • Configurable logging with file rotation for security and compliance.

Pulumi Infrastructure on AWS

  • Automated provisioning of AWS resources including VPC, subnets, RDS, and EC2 instances.
  • Detailed documentation and setup guides for deployment consistency.
  • Secure setup of Internet Gateway and route tables for reliable network connectivity.

Serverless Lambda Function

  • Handling of assignment submissions with real-time processing and feedback.
  • Integration with AWS SNS, DynamoDB for event logging, and external services like Mailgun for email notifications.

Technologies Used

Node.js, AWS, Pulumi, MySQL, Lambda, SNS, DynamoDB, Mailgun, Google Cloud Storage.

Challenges and Learnings

The integration of diverse technologies and the assurance of a seamless flow of data across different components presented significant challenges. This project emphasized the importance of security practices and thorough system testing to maintain integrity and performance.

Conclusion

This project not only highlights technical proficiency but also demonstrates the capability to architect and deploy a highly available, secure, and scalable system using modern web technologies. It exemplifies a holistic approach to application development from initial coding to infrastructure management and serverless computing.
